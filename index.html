Hola: Mi Primer Index.HTML

En este HTML se muestra información de algunas de las etiquetas más usadas de HTML. Se hace usa de las etiquetas de carácter, para poder mostrarlas, ya que en este formato el interprete de HTML no las ejecuta.

Las entidades de carácter hacen referencia a un caracter, por ejemplo, para refererirno a < y > se usa & lt y & gt, claro pegados. Adentro, de estas entidades, se pueden poner los editificadores de las etiquetas, para que no sean ejecutadas

Una etiqueta en HTML tiene la siguiente estructura

<etiqueta args> </etiqueta>

Entre las etiquetas a usar, se encuentra (se omite la de cierre):

  • < b >: Pone el texto en negrita
  • < i >: Pone el texto en cursiva o itálica
  • < u >: Pone el texto subrayado
  • < br >: Salton de
    línea
  • < !DOCTYPE HTML >: Se usa para indicar el tipo de documento.
  • < html >: todo lo que se encuentra dentro está etiqueta sera código html. Se conoce como elemento raíz
  • < title >: contiene el nombre que aparece en la pestaña del navegador web
  • < head >: contiene el código de la etiqueta title y todo lo que no se va mostrar en la pagina. También, puede contener un resumén de la página que puede aparecer en su búsqueda, como código CSS
  • < body >: contiene todo lo que se va a mostrar en la página
  • < hX >: con X desde 1 hasta 6, contiene textos en negrilla que disminuye en tamaño con el aumento de X
  • < p >: contiene textos o en linea, que se pueden poner en negrita, cursiva, subrayado, etc.

    Además, se le puede adicionar los argumentos align = "left/center /right", para alinear al texto, este argumento también se puede usar en otras etiquetas que encierran texto, como son las < hX >
  • < ul >: contiene listas
  • < li >: contiene los elementos de la lista y adiciona, un circulo como itemizador

Etiqueta (p)(\p) para poner texto, se adiciona align = "left", para alinear a la izquierda.

Etiqueta (p)(\p) para poner texto, se adiciona align = "center", para alinear a la izquierda.

Etiqueta (p)(\p) para poner texto, se adiciona align = "right", para alinear a la derecha.
